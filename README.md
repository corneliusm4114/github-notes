### Github
Its like the website version of Github where you can use Git and its commands on a formal website. Its good for collaborating on projects big or small and is used as a social website which is managed through Git commands.

### Git
Git is the main version control and the application thats behind the scenes in  Github. It mangaes the main collaboration Github has while Github expands on it.

### Repo
Short for repository pretty much a folder where you can put files in and represents a subject of the directory you created.

### Commit
Pretty much the save button for any changes you made in Git(hub), good for saving your work and use in tandeom with GIT to look back at your changes.
